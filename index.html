🌾 Controller Setup

agri.begin()

Initialize the Agri controller. Waits for button press to start.

agri.end()

Gracefully stop the program and restart.

agri.wait(ms)

Pause execution for ms milliseconds.

⚙️ Motor Control

agri.set_motor_power(ch, pwr)

ch: 1–4 | pwr: –100 to 100

agri.set_motor_powers(p1, p2, p3, p4)

Set all four motors at once.

agri.set_motor_speed(ch, spd)

ch: 1–4 | spd: –720 to 720 deg/s

agri.set_motor_target(ch, spd, tgt)

Move to encoder target position.

agri.stop_motor(ch)

Stop a single motor channel.

agri.stop_all_motors()

Stop all motor channels immediately.

agri.set_motor_brake(ch, mode)

True = brake, False = coast

agri.reset_encoder(ch)

Reset encoder count to zero.

agri.get_encoder_count(ch)

Returns raw encoder count.

agri.get_encoder_degrees(ch)

Returns position in degrees.

🔩 Servo Control

agri.set_servo_position(ch, angle)

ch: 1–8 | angle: 0–180°

agri.disable_servo(ch)

Disable PWM output for servo channel.

📡 Sensors

agri.read_ultrasonic(port)

Returns distance in centimeters.

agri.read_line_sensor(port)

Returns 0 (light) or 1 (dark).

agri.read_analog(port)

Returns 0–4095 (12-bit ADC).

agri.read_digital(port)

Returns 0 or 1.

agri.read_soil_moisture(port)

Returns moisture % (0–100).

agri.read_light(port)

Returns light level (0–4095).

agri.read_temperature(port)

Returns temperature in °C.

💡 Lights & Sound

agri.set_led(r, g, b)

Set onboard RGB LED color.

agri.led_off()

Turn off onboard LED.

agri.play_tone(freq, dur_ms)

Play a tone at frequency for duration.

agri.play_note(note, dur_ms)

note: "C4", "G3", etc.

agri.beep()

Short confirmation beep.
